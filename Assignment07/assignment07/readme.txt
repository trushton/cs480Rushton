To Compile the Program:
----------------------
1) Open a terminal and change to the assignment folder.
2) Navigate to the "build" folder. (cd build)
3) Once in build, simply type "make".


To Run the Program:
------------------
1) Navigate to the "bin" folder. (cd ../bin)
2) Once in bin, simply type "./SolarSystem"


Program Controls:
----------------

	w : move forwards
	s : move backwards
	a : move left
	d : move right
	q : move down
	e : move up

	i : look up
	j : roll left
	k : look down
	l : roll right
	u : look left
	o : look right

	- : decrease camera movement speed
	= : increase camera movement speed

	Press keys 1-9 to follow a planet
	1 : Mercury
	2 : Venus
	3 : Earth
	4 : Mars
	5 : Jupiter
	6 : Saturn
	7 : Uranus
	8 : Neptune
	9 : Pluto

	Right Mouse Click : open menu

  	ESC - closes the program


Menu Options:
------------
Start Orbit - Makes the planets continue orbiting
Stop Orbit - Makes the planets stop orbiting
Change Scale - Change between realistic and scaled data
Toggle Orbit Paths - Toggle whether orbit paths are drawn or not
Quit - Closes the program.
